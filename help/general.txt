awsdo - AWS CLI wrapper utility

USAGE:
    awsdo <command> [options]

COMMANDS:
    init            Initialize AWS CLI, SSM plugin, and AWS SSO profile
    login           Log in to AWS SSO
    get-credentials Get AWS credentials (access key, secret key, session token)
    credentials     Alias for get-credentials
    instances       Manage EC2 instances (find, list, add, remove)
    terminal        Start an SSM terminal session to an EC2 instance
    bastion         Start a port forwarding session through a bastion host
    bastions        Manage bastion hosts (list, add, update, remove)
    profiles        Manage AWS CLI profiles (add, list, remove)
    repl            Start interactive REPL mode
    docs            Display the full documentation in a web page
    help            Show help information
    version         Show application version

    # Linux-style commands (alternative syntax)
    ls, list    List instances, bastions, or profiles (e.g., 'ls instances', 'list bastions', 'ls profiles')
    add         Add an instance, bastion, or profile (e.g., 'add instance', 'add bastion', 'add profile')
    update      Update a configured instance or bastion (e.g., 'update instance', 'update bastion')
    rm          Remove an instance, bastion, or profile (e.g., 'rm instance', 'rm bastion', 'rm profile')
    find        Find instances (e.g., 'find instance')

For detailed help on a specific command, use:
    awsdo help <command>
