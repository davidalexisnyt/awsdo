awsdo get-credentials - Get AWS credentials for a profile

USAGE:
    awsdo get-credentials [--profile <aws cli profile>]
    awsdo get-credentials [-p <aws cli profile>]
    awsdo credentials [--profile <aws cli profile>]
    awsdo credentials [-p <aws cli profile>]

DESCRIPTION:
    Retrieves and prints AWS credentials (access key, secret key, and session
    token) for the specified profile in JSON format. If no profile is specified,
    uses the default profile from configuration. The credentials can be used by
    other applications that require AWS SDK authentication.

    The output is in JSON format:
    {
        "Version": 1,
        "AccessKeyId": "...",
        "SecretAccessKey": "...",
        "SessionToken": "...",
        "Expiration": "..."
    }

OPTIONS:
    --profile, -p    AWS CLI profile to use for credentials

EXAMPLES:
    awsdo get-credentials --profile dev
    awsdo credentials -p production
